# Deep Bayesian Active Learning with Image Data
This repository aims at reproducing the results of the paper "Deep Bayesian Active Learning with Image Data" (https://arxiv.org/abs/1703.02910) using JAX/Flax.

We will present:
- Short introduction to the claims and the theory of the paper
- Reproduce experimental results: training and evaluation of Bayesian CNNs
- Implement and evaluate active learning strategies
- Evaluate the performance of the active learning strategies
